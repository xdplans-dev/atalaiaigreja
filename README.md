# Igreja Atalaias Vale da Benção

## Descrição
Este projeto foi desenvolvido por David Xavier, através da XD Plans, em parceria com a Igreja Atalaias Vale da Benção.

A proposta deste projeto é criar uma plataforma digital moderna, elegante e estratégica para fortalecer a presença online da igreja, apoiar o crescimento ministerial, facilitar o acesso às informações, divulgar cultos, eventos, pedidos de oração, discipulado e aproximar pessoas da mensagem do Evangelho.

O projeto também nasce com uma visão maior: unir fé, conhecimento e tecnologia para abrir caminhos de crescimento para jovens e adultos, incentivando aprendizado, desenvolvimento digital, organização ministerial e novas oportunidades para um futuro melhor.

A parceria entre David Xavier, XD Plans e a Igreja Atalaias Vale da Benção busca construir não apenas um site, mas uma plataforma viva, escalável e preparada para evoluir com o tempo.

## Objetivos do projeto
• fortalecer a presença digital da igreja
• divulgar a missão, visão e propósito ministerial
• facilitar o contato de visitantes
• organizar informações sobre cultos e eventos
• apoiar o discipulado e a trilha ministerial
• incentivar jovens e adultos a conhecerem tecnologia
• criar uma base digital que possa crescer continuamente
• aproximar pessoas da igreja e do Evangelho

## Tecnologias utilizadas
• React
• Vite
• TypeScript
• TailwindCSS
• Framer Motion (motion/react)
• SEO Metadata
• React Router

## Visão de Futuro
Este projeto poderá evoluir futuramente para uma plataforma completa, com painel administrativo, gerenciamento de eventos, pedidos de oração, área de membros, trilha de discipulado, conteúdos para jovens e adultos, integração com redes sociais, transmissões ao vivo e aplicativo mobile.

A ideia é que a tecnologia seja usada como ponte para conhecimento, crescimento espiritual, desenvolvimento pessoal e impacto positivo na comunidade.
